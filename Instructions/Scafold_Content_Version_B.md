---

## HOW TO USE THIS DOCUMENT

For each of the 37 posts, you'll find:

1. **3 HOOK OPTIONS** — The opening 1-2 lines that stop the scroll. Each has a label, the hook text, and a breakdown of the psychological angle it uses.
2. **VERSION B (FULL POST)** — A complete alternate version built on a different angle than Version A. Same message, different entry point.
3. **PLATFORM OPTIMIZATION** — Specific notes on how this post should differ between LinkedIn and Instagram (length, tone, visual direction, CTA placement).

**Decision rule:** If Version A uses a *story-driven* hook, Version B uses a *data/contrarian* hook (and vice versa). This gives you maximum range when testing.

## PLATFORM-SPECIFIC OPTIMIZATION CHEAT SHEET

### LinkedIn vs. Instagram — The Core Differences

| Element | LinkedIn | Instagram |
|---------|----------|-----------|
| Hook style | Data, contrarian takes, frameworks | Emotional, visual, relatable |
| Length | 150-400 words | 80-150 words caption |
| Tone | Professional vulnerability | Casual authenticity |
| CTA | "→ scafold.ai" at end | "Link in bio" — always |
| Hashtags | 2-4, end of post | 5-8, mix of broad and niche |
| Format | Text-first, line breaks matter | Visual-first, caption is secondary |
| Best hooks | Numbers, "Here's what nobody tells you", "I learned X the hard way" | Questions, dares, "This is you" statements |
| Engagement driver | Comments (debates, personal stories) | Saves and shares (saveable insights) |
| Peak times | 7-10 AM ET (Tue-Thu strongest) | 9-11 AM or 6-8 PM ET |

### Cross-Posting Rules

1. **Never copy-paste between platforms.** Rewrite for each.
2. **LinkedIn leads with insight.** Instagram leads with emotion.
3. **LinkedIn posts can be long.** Instagram captions should be front-loaded — the hook must land before the "more" fold (first 125 characters).
4. **Instagram needs a visual concept for every post.** LinkedIn is text-only.
5. **Hashtag strategy differs.** LinkedIn: 2-4 targeted (#WhatBugsYou #scafold #buildinpublic). Instagram: 5-8 including discovery tags (#startuptips #entrepreneurlife #aitools).
6. **CTAs differ.** LinkedIn can include URLs inline. Instagram must always say "Link in bio."
7. **Same message, different entry point.** If LinkedIn version uses a data hook, Instagram version uses an emotional/visual hook. And vice versa.
